# BTCPay Agentbot

BTCPay Server plugin for Agentbot platform — accept BTC/crypto payments on agent subscriptions.

## What It Does
- Accept Bitcoin & Lightning payments for Agentbot subscriptions
- No third-party processors, no fees
- P2P settlement on Base + Bitcoin
